Image processing using PIL and NumPy

Posted on 2020-04-11 | In Computer-science | | Visitors:

By reading the images by PIL-Image module and storing as NumPy-ndarray, we can conveniently process the images.

RGB (color) images become 3D ndarray (row (height) x column (width) x color (3)), and black and white (grayscale) images become 2D ndarray (row (height) x column (width)).

Range: $0-255$, each dimension.
